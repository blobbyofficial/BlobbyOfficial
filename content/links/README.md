# Links hub content

`/links/` renders from `content/links/links.json`.

## Edit links
- Update top profile text with `name` and `bio`.
- Update quick social chips in `social`.
- Update primary Linktree-style buttons in `links`.

## Link object shape
- `title`
- `subtitle`
- `url`

Use absolute URLs (`https://...`), internal paths (`/page/`), or `mailto:`.
